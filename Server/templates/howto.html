How to Use the AWOS Website


The occupancy status of seating throughout Building 11 is given in the "seating" page of this website. There occupancy information relating to each seat is given. The information is given as three columns Location, Occupied, Date. The information in these main categories can be sorted by clicking on their respective headings.

  • Location is both the name and location of the desk given in UTS Style
    CB11 - Building Number
    XX - Floor Number
    XXX - Desk/Room Number
  • Occupied is the occupancy status of the desk. False indicates the desk is unoccupied, whereas True indicates the desk is occupied and unable to be used.
  • Date is the last time at which the seat status was updated, ideally this should be within the past 5-10 minutes, however in the event of a system issue, this time may be increased.
  • Page Last Updated is the time at which this page was last loaded by you (the user)'s computer. The site will automatically update every 5 minutes, however this does not necessarily mean that there has been any change to the occupancy status of any of the desks, just that your system has the most recent information available.
